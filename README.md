## üöÄ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- MongoDB
- Redis
- Docker (optional)
- API Keys for AI services

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/deeplearn.git
cd deeplearn
```

2. Install dependencies:

```bash
# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install
```

3. Configure environment variables:

```bash
# Backend
cp .env.example .env
# Frontend
cp .env.example .env
```

4. Start development servers:

```bash
# Start backend
cd backend
npm run dev

# Start frontend
cd frontend
npm run dev
```
